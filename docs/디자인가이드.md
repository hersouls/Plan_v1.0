## 1. 디자인 토큰 시스템

### 1.1 토큰 철학
- **Single Source of Truth**: 모든 디자인 결정의 중앙 집중화
- **Platform Agnostic**: CSS, React, 모바일에서 동일하게 사용
- **Hierarchical Structure**: Foundation → Semantic → Component 토큰 구조
- **Automated Consistency**: 토큰 변경 시 모든 인스턴스 자동 반영
- **Design System Constraints**: 제약된 디자인 토큰으로 일관성 확보
- **Responsive by Default**: 모바일 우선 반응형 설계
- **Performance Optimized**: PurgeCSS로 미사용 스타일 제거
- **Icon-Only Interface**: 텍스트 없이 아이콘만으로 명확한 의미 전달
- **No Emoji Policy**: 이모지 사용 금지, 전문적인 아이콘 라이브러리만 사용
- **Premium Components First**: Tailwind UI 프리미엄 컴포넌트 우선 활용

```tsx
  children: React.ReactNode;
  className?: string;
}

const Text: React.FC<TextProps> = ({ variant, children, className }) => {
  const styles = TextStyleTokens[variant];
  
  return (
    <span 
      className={cn('font-pretendard', className)}
      style={styles}
    >
      {children}
    </span>
  );
};
```

### 4.2 스페이싱 적용 유틸리티

```tsx
// 스페이싱 토큰 기반 레이아웃 컴포넌트
interface StackProps {
  gap: keyof typeof SpacingTokens.semantic;
  direction?: 'row' | 'column';
  align?: 'start' | 'center' | 'end' | 'stretch';
  justify?: 'start' | 'center' | 'end' | 'between' | 'around';
  children: React.ReactNode;
}

const Stack: React.FC<StackProps> = ({ 
  gap, 
  direction = 'column', 
  align, 
  justify,
  children 
}) => {
  const gapValue = SpacingTokens.semantic[gap];
  
  return (
    <div 
      className={cn(
        'flex',
        direction === 'row' ? 'flex-row' : 'flex-col',
        align && `items-${align}`,
        justify && `justify-${justify}`
      )}
      style={{ gap: gapValue }}
    >
      {children}
    </div>
  );
```

```tsx
const GlassCard: React.FC<GlassCardProps> = ({
  variant = 'medium',

  children,
  className,
  onClick
```
